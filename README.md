# Installing

Make sure Python3 is installed, and pip is installed for Python3

Note: use pip or pip3, whichever is associated with your Python3

Then:
```
pip install numpy
pip install scipy
pip install sklearn
pip install pandas
pip install matplotlib
pip install seaborn
pip install biopython
```

For ubuntu, also:
```
sudo apt install python3-tk
```
